#使用YCSB测试LevelDB的性能

使用LevelDB的接口连接至YCSB，与YCSB的接口连接实现测试性能。

LevelDB接口中文介绍

YCSB接入新的数据库的方法

最后在GitHub上找到了使用YCSB1.4.0版本的ycsb-leveldb

使用命令

mvn -pl com.yahoo.ycsb:leveldb-binding -am clean package

对文件进行编译

之后由于此文档中把ycsb生成了ycsb.sh程序

因此使用 ./ycsb进行执行,测试leveldb性能
